# mealy-legal

Public Privacy Policy and Terms of Service for the [Mealy](https://github.com/ray-co-ai/mealy) iOS and Android app.

Hosted via GitHub Pages from the `main` branch.

- Privacy: <https://ray-co-ai.github.io/mealy-legal/privacy/>
- Terms: <https://ray-co-ai.github.io/mealy-legal/terms/>

Editing: change `privacy.md` or `terms.md`, bump the "Last updated" date, push to `main`. GitHub Pages rebuilds in a minute or two.

The app references these URLs from `src/lib/mealyLegalUrls.ts` in the Mealy repo - keep them in sync if you ever move the pages.
